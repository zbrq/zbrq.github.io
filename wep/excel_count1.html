表格基本常识

表格处理数据一般用到的技能：1.计算，2.统计，3.筛选，4.引用，5.展示和美化，6.录入

学习表格的终极目标：使表格更方便，使表格功能更实用，使表格功能更全面，简化流程，提高个人效率

1.新建的EXCEL表格文件不能称之为表，只能称之为工作簿，就是像一个本子，而EXCEL文件里面的sheet1那些表页面才能叫作“表”，表就是一个工作簿里面的每个页面。

2.关于引用的理解：

引用有3种：
1.相对引用（这个变，另一个也变）《跟随变化》
        D2=B2*C2
        D3=B3*C3
2.绝对引用 (拖动时不会跟随变化)《拖动时绝对固定不变》按F4+Fn键固定单元格数据不变化$F$2
        E2=D2*$F$2
        E3=D3*$F$2
        E4=D4*$F$2
3.混合引用 (只固定行/列 ，拖动时行号跟着变化 列标不跟随变化)$A3
        D3=$A3*B3
        E3=$A3*C3

    相对引用：
            逐行逐列引用     D2
    绝对引用:
            固定行和列引用   $D$2
    混合引用:   
            列固定  $D2
            行固定  D$4

2）筛选函数

查找利器：lookup

联动查找神器：VLookup     XLOOKUP

查找中的黄金组合: index + match

查找引用函数详解：
  1.lookup()  格式：lookup(你想找的,在这个数据范围中找)
          例子：LOOKUP(E2,A2:C12)
    lookup() 实现按照多条件查找
          格式：
------------------------
  2.vlookup()   格式： vlookup(你想找的,查找范围,找到后想要哪列的东西,准确找还是近似找)
------------------------
  3.hlookup()   格式： hlookup(你想找的,查找范围,找到后想要哪行的东西,准确找还是近似找)
            准确找为： 0 或 false
            近似找为： 1 或 true              
------------------------
  4.Xlookup()   格式： Xlookup(想查找,查找范围,返回值数组,未找到值,匹配模式,搜索方式)
        --返回值数组：返回的数据或范围
        --未找到值：未匹配到时的返回值
        --匹配模式: 0 完全匹配 ，未找到时返回 #N/A
                   1 完全匹配 ，未找到时返回较大的项   
                  -1 完全匹配 , 未找到返回较小项
                   2 通配符进行匹配
        --搜索方式: "1" & "-1" 正反向匹配
                   "2"依赖looup_array升序搜索
                   "-2"依赖looup_array降序搜索

3）计算函数

求和      sum     sumif     sumifs    

求平均     average     averageif

求最大值和最小值     max

  • 万能求和之王:sumproduct 函数

    格式：
    1.普通求和： sumproduct(求和区域)
    2.单条件求和：sumproduct((条件区域=条件)*(要求和的数据范围))
    3.多条件求和：sumproduct((条件区域1=条件1)*(条件区域2=条件2)...*(要求和的数据范围))

  • 汇总型函数 substotal 

    功能： 汇总型函数，将平均值，技术，最大最小，相乘，标准差，求和，方差等参数化，只要会这个函数，求和函数都可以抛弃！
  格式： substotal(引用范围,参数)

  • 取整函数 int     按小数位取数round

    例子：round(3.1415,2)=3.14  
     round(3.1415,1)=3.1

表与表之间的处理

EXCEL表格——数据引用

数据引用有两种形式:      
    1.表与表之间引用数据（引用单列数据 引用多列数据）
    2.工作簿与工作簿之间的表引用数据（引用单列数据 引用多列数据）
设计表格时通常考虑减少工作量，提高工作效率，所以会使用自动化表格，让数据相互关联
--------

1.Vlookup  2.Xlookup  3.offset 
函数使用详解：
    格式：
    offset(单元格或区域,走几行《正数向下负数向上》,走几列《正数向右负数向左》,高度,宽度) //宽度高度只能是正数,省略时视为1，与区域一样
